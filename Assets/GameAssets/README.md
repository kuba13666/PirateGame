# Game Assets

This folder contains the original PNG files used in the game:

- **Ship.png** - Player ship sprite
- **enemy_giant_crab.png** - Crab enemy (1 HP, common)
- **enemy_harpy.png** - Harpy enemy (2 HP, uncommon)
- **enemy_mermaid.png** - Mermaid enemy (3 HP, rare)

These files are copied to `Assets/` folder in Unity for use in the game.
